# -Analyse-des-Lois-de-Finances-du-Cameroun
Plateforme d'analyse budgétaire et sémantique des lois de finances du Cameroun. Elle mesure la conformité des allocations budgétaires avec les piliers de la SND30 (Stratégie Nationale de Développement 2030).  Lien de l'application streamlit : https://analyse-loi-des-finances-cameroun.streamlit.app/
